# Visualization-Tool-for-Electric-Vehicle-Charge-and-Range-Analysis
EV Charging Stations Data Analysis (Tableau)
Project Overview

This project analyzes Electric Vehicle (EV) Charging Stations in India using data visualization techniques. The objective of this project is to understand the distribution, availability, and regional patterns of EV charging infrastructure across the country.

Using Tableau, interactive dashboards and visualizations were created to identify trends, regional differences, and infrastructure gaps in EV charging networks.

Objectives

Analyze the distribution of EV charging stations across different regions in India.

Identify regions with the highest and lowest charging infrastructure.

Visualize patterns and clusters of charging stations geographically.

Provide insights that can help in future EV infrastructure planning.
